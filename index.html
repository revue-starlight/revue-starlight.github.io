什么是尺取？

尺取是一个区间内的需要再连续段满足某个性质的情况下求解某个最优解（通常是最短区间之类）

题意

一群人看电影，有一些位置被占据，剩下的人要尽可能的坐的靠近。给出空的位置矩阵，坐的尽可能近的衡量标准是在一群人的位置最外面套一个最小矩形，这个矩形的面积。
电影院行列皆小于300，人数小于300*300,保证有解。

思路

行列皆小于300，枚举多少行，从1—300是 O(n)。枚举从第几行开始，尺取共计
$O(n^3)$

问题复杂在二维前缀和的处理上。还有要保证如果某一行的有效座位大于k人，会出现l>r的情况，此时要l++并重新计算sum

二分答案/二分专题

模板

origin : 模板源自去年上课，但是tourist也是这么写的。所以非常好。

右边是可行区间

1. l,r边界
保证r时一定可以取到。否则会返回r，尽管r没有用
mid=(l+r)/2;
if (ok) r=mid; else l=mid+1
若不能保证r一定可以取到，则取r+1为r边界。这样如果返回r+1，则是全区间无解。
l必为不可行或最优可行，不然会存在比l小的更优解被忽视。也就是没有比l小的更优解即可

2. mid
mid指针指向不可行区间一侧，mid不可行就踢过来，否则r=mid;

右边是可行区间

同上

形如lower_bound的二分

lowerbound返回arr[i]中第一个大于/等于x的指针。
不难看出是找大于等于的最左端，也就是可行区间在右侧。
check()函数判断这个值是否大于等于x。
⭐有可能全区间无解。此时可行区间侧指针+1。

C.Number

数论/Cu

题意

初始有n个数字，第i个数字为 $a_i$ 。
每次操作，可以选择一个数字x，把它修改为 $x^2$或者 $\lfloor \sqrt{x} \rfloor$。
最少需要操作几次，才能让所有数字都相等？

解法

铜牌题，首先考虑一个平方数平方开根号等于没做，不断平方也不可能收束到一个数。故往开方考虑，开方到最后肯定开到1。而开方再平方会到另外一个等价类里面去。嘛。。等价类感官上更像一点，就是令某个数的等价类为x,x^2,x^4….。而意识到复杂度之后，☆复杂度☆ $a_i < 1e5$ ，$n < 1e5$而且一个数开3次方or平3次方基本上就越界了，那所有的数字可以在小常数时间内搞出他到 所有 可达数的位置。大概是10次吧。
考虑开一个map,其实数组也行。记录这个可达数可以被多少数字到达，再开一个map，记录这个可达数的总步数。
